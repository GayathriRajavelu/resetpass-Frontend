                                             # Password Reset Frontend

## Overview
This is the frontend component for the password reset flow in the FSD-B-33 project.

## Features
- User-friendly password reset interface
- Form validation
- Secure token handling
- Success/error messaging

## Installation
```bash
npm install
```

## Usage
```bash
npm start
```

## Technologies
- React
- CSS/Styled Components
- Axios (or fetch API)

## Project Structure
```
├── components/
├── pages/
├── utils/
└── styles/
```

## Environment Variables
Create a `.env` file:
```
REACT_APP_API_URL=your_api_url
```
